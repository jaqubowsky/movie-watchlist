### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jaqubowsky/movie-watchlist.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
